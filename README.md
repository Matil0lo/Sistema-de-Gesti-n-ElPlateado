# Sistema-de-Gestion-ElPlateado
Proyecto de gestión y administración desarrollado con HTML, CSS, JavaScript y jQuery.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- jQuery
- localStorage

##  Descripción
Sistema web para la gestión y administración de productos, stock y pedidos.  
Incluye módulos para clientes y sucursales, con control de inventario y registro de movimientos.
